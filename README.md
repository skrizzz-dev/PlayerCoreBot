# PlayerCoreBot - Рекомендации игр

[![Python](https://img.shields.io/badge/Python-3.8+-blue)](https://python.org)
[![aiogram](https://img.shields.io/badge/aiogram-3.x-orange)](https://github.com/aiogram/aiogram)
[![Telegram](https://img.shields.io/badge/Telegram-Bot-purple)](https://t.me/PlayerCoreBot)

Простой Telegram-бот на **aiogram** для персональных рекомендаций компьютерных игр. Анализирует предпочтения пользователя и подбирает игры из базы данных Steam/IGDB.

## Установка

```bash
git clone https://github.com/skrizzz-dev/PlayerCoreBot.git
cd PlayerCoreBot
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или на Windows: venv\Scripts\activate
pip install -r requirements.txt
```
